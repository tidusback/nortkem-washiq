## 🚀 Deploy to Vercel (Step-by-Step)

### Option A — Drag & Drop (Easiest, no coding needed)

1. **Install Node.js** from https://nodejs.org (download the LTS version)

2. **Open Terminal** (Mac) or **Command Prompt** (Windows)
   - Navigate to this folder:
     ```
     cd path/to/nortkem-washiq
     ```

3. **Install dependencies:**
   ```
   npm install
   ```

4. **Build the app:**
   ```
   npm run build
   ```
   This creates a `dist/` folder.

5. **Go to https://vercel.com** → Sign up free with GitHub or email

6. **Deploy:**
   - Click **"Add New Project"**
   - Click **"Deploy from CLI"** or drag the `dist/` folder into the Vercel dashboard
   - Done! You get a live URL like `nortkem-washiq.vercel.app`

---

### Option B — GitHub + Vercel (Auto-deploys on every update)

1. Create a free account at https://github.com
2. Create a new repository called `nortkem-washiq`
3. Upload all files in this folder to that repo
4. Go to https://vercel.com → Import your GitHub repo
5. Vercel auto-detects Vite → click **Deploy**
6. Every time you push changes to GitHub, Vercel auto-rebuilds

## 📱 Add to Home Screen (Mobile)

Once deployed, open the URL on your phone:
- **iPhone:** Tap Share → "Add to Home Screen"
- **Android:** Tap ⋮ menu → "Add to Home Screen"

It will work like a native app — full screen, no browser bar.
